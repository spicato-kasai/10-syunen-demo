# Box Game

このプロジェクトは、四角の箱を表示するシンプルなゲームです。ユーザーは箱を操作し、小石を箱に入れることができます。

## ファイル構成

- `src/index.html`: アプリケーションのHTMLエントリーポイント。箱を表示するための基本的な構造を持っています。
- `src/styles/main.css`: 四角の箱やその他の要素のスタイルを定義するスタイルシート。
- `src/scripts/main.ts`: DOMを操作して箱を表示し、ゲームの初期設定を行うメインスクリプト。
- `src/scripts/game.ts`: 箱の動きや小石を箱に入れるための機能を実装するゲームロジック。
- `src/scripts/types/index.ts`: ゲーム内で使用するインターフェースや型を定義するTypeScriptの型定義。
